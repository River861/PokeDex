## 简介

一个通过问答识别精灵宝可梦(Pokémon)的专家系统

基于框架、不确定性管理、决策树



## 运行方法

```
python3 app.py
```

按工具栏搜索键 [开始/重新] 搜索；按记录键 [增添/覆盖] 宝可梦信息到图鉴中。
